## 🗄️ النماذج المضافة (7 Models)

### 1. ⚙️ Account - الحسابات المحاسبية
**الجدول:** `accounts`  
**الهدف:** دليل الحسابات الموحد (Chart of Accounts)

**الميزات:**
- دعم الحسابات الهرمية (Parent-Child)
- 5 أنواع رئيسية: أصول، خصوم، حقوق ملكية، إيرادات، مصروفات
- تتبع الأرصدة تلقائياً
- تفعيل/تعطيل الحسابات

**الأكواد القياسية:**
```
1000-1999: الأصول (Assets)
2000-2999: الخصوم (Liabilities)
3000-3999: حقوق الملكية (Equity)
4000-4999: الإيرادات (Revenue)
5000-5999: المصروفات (Expenses)
```

---

### 2. 📝 JournalEntry - القيد المحاسبي
**الجدول:** `journal_entries`  
**الهدف:** تسجيل جميع العمليات المحاسبية

**الميزات:**
- ترقيم تلقائي فريد (JE-2026-001)
- ربط مع المصدر (فاتورة، دفعة، مصروف، راتب)
- نظام الاعتماد (Posted/Unposted)
- القيود المعتمدة لا يمكن تعديلها (Audit Trail)

---

### 3. 📋 JournalLine - سطر القيد
**الجدول:** `journal_lines`  
**الهدف:** تفاصيل القيد (مدين/دائن)

**القاعدة الذهبية:**
```
مجموع المدين = مجموع الدائن
Sum(Debit) = Sum(Credit)
```

---

### 4. 🧾 Receipt - سند القبض
**الجدول:** `receipts`  
**الهدف:** توثيق المقبوضات من الطلاب

**الميزات:**
- ترقيم فريد (REC-2026-001)
- المبلغ بالأرقام والحروف
- ربط مع القيد المحاسبي
- تسجيل طريقة الدفع والمرجع

**الاستخدامات:**
- استلام الأقساط الدراسية
- استلام رسوم التسجيل
- استلام أي مستحقات أخرى

---

### 5. 💰 PayrollSheet - مسير الرواتب
**الجدول:** `payroll_sheets`  
**الهدف:** إدارة رواتب الموظفين شهرياً

**الميزات:**
- مسير شهري شامل لجميع الموظفين
- حالات: مسودة، معتمد، مدفوع، ملغي
- دعم WPS (نظام حماية الأجور الإماراتي)
- توليد ملف SIF تلقائي
- ربط مع القيود المحاسبية

**دورة العمل:**
```
Draft → Approved → Generate SIF → Bank Transfer → Paid
```

---

### 6. 📊 PayrollItem - عنصر الراتب
**الجدول:** `payroll_items`  
**الهدف:** تفاصيل راتب كل موظف

**المكونات:**
```
الراتب الأساسي (Basic السلاري)
+ بدل السكن (Housing Allowance)
+ بدل المواصلات (Transport Allowance)
+ بدل الهاتف (Phone Allowance)
+ بدلات أخرى (Other Allowances)
= إجمالي الراتب (Gross Salary)

- التأمينات (Insurance)
- القروض (Loans)
- استقطاعات أخرى (Other Deductions)
= صافي الراتب (Net Salary)
```

**حقول WPS:**
- اسم البنك
- رقم IBAN
- لتوليد ملف SIF

## 🎓 المبادئ المحاسبية المطبّقة

### 1. ✅ مبدأ القيد المزدوج (Double Entry)
```
كل عملية مالية = قيد يحتوي على:
- جانب مدين (Debit)
- جانب دائن (Credit)
- المدين = الدائن دائماً
```

**مثال:**
```
استلام 1,500 درهم من طالب:
مدين: حساب البنك         1,500
    دائن: حساب المدينين  1,500
```

---

### 2. ✅ مبدأ الاستحقاق (Accrual Basis)
```
تسجيل الإيرادات عند استحقاقها (وليس عند تحصيلها)
تسجيل المصروفات عند حدوثها (وليس عند دفعها)
```

**مثال:**
```
عند تسجيل طالب:
  الإيراد مستحق ← يُسجل فوراً
  القبض الفعلي ← يُسجل لاحقاً عند الاستلام
```

---

### 3. ✅ المعادلة المحاسبية
```
الأصول = الخصوم + حقوق الملكية
Assets = Liabilities + Equity
```

---

## 🔄 دورة القيد المحاسبي النموذجية

```
1. وقوع العملية المالية
   ↓
2. إنشاء JournalEntry مع رقم فريد
   ↓
3. إضافة JournalLines (مدين + دائن)
   ↓
4. التحقق: Sum(Debit) = Sum(Credit)
   ↓
5. الاعتماد (isPosted = true)
   ↓
6. تحديث أرصدة الحسابات (Account.balance)
   ↓
7. توليد التقارير (TrialBalance, Income Statement)
```
