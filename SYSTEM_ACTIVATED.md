## 🔧 كيفية الاستخدام

### 1. تسجيل تسليم طالب
1. اختر تاريخ ووقت في عمود "تاريخ التسليم"
2. اضغط زر 💾
3. ✅ يتم إنشاء submission جديد
4. الحالة تتغير من 🔴 إلى 🔵

### 2. تصحيح واجب
1. أدخل اسم المصحح (مثال: د.محمد)
2. اختر تاريخ التصحيح
3. أدخل الدرجة (0-100)
4. اضغط زر 💾
5. ✅ يتم حفظ التصحيح
6. الحالة تتغير إلى 🟢

### 3. البحث والفلترة
- **البحث**: اكتب اسم أو رقم الطالب
- **الفلتر**: اختر الحالة من القائمة (الكل / لم يسلم / سلّم / قيد التصحيح / صُحح)
