## 🔍 المنطق المُطبق

### 1. جلب الطلاب
```javascript
GET /api/v1/academic/units/:unitId/students
→ يرجع فقط الطلاب المسجلين في هذه الوحدة
```

### 2. جلب التسليمات
```javascript
GET /api/v1/assignments/:id/submissions
→ يرجع التسليمات الموجودة
```

### 3. الدمج
```javascript
كل طالب مسجل = صف في الجدول
إذا كان له submission → يملأ البيانات
إذا لم يكن له submission → الحالة "missing"
```

### 4. الحالات
```javascript
missing → لا يوجد submission
submitted → يوجد submission لكن لا يوجد درجة
being_assessed → يوجد assessor لكن لا درجة
graded → يوجد درجة
```

## ⚠️ إذا ظهرت رسالة "لا يوجد طلاب"

**السبب**: الوحدة ليس لها فصول مسجلة أو الفصول لا تحتوي طلاب.

**الحل**:
1. تحقق: هل الوحدة مربوطة ببرنامج؟
2. تحقق: هل البرنامج له فصول؟
3. تحقق: هل الفصول بها طلاب مسجلين؟

**التحقق من قاعدة البيانات**:
```sql
-- 1. ربط الوحدة بالبرنامج
SELECT * FROM program_units WHERE unit_id = 'UNIT_ID';

-- 2. الفصول
SELECT * FROM classes WHERE program_id = 'PROGRAM_ID';

-- 3. الطلاب
SELECT * FROM student_enrollments WHERE class_id = 'CLASS_ID' AND status = 'enrolled';
```
